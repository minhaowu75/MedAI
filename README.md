# MedAI

## Description

MedAI is an AI Medical Agent that answers user questions about symptoms, test results, and health concerns.

Users can easily ask the chatbot questions, list symptoms, and give test results and MedAI will ask elaborating questions and give thoughts on what could be going on.

Claude was used to assist in debugging the code.

## NOTICE

***The information provided on MedAI is for general knowledge and informational purposes only, and does not constitute medical advice. It is essential to consult with a qualified healthcare professional for any health concerns or before making any decisions related to your health or treatment.***
